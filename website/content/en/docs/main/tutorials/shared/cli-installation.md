1. Install ClusterLink CLI on Linux or Mac using the installation script:

    ```sh
    curl -L https://github.com/clusterlink-net/clusterlink/releases/latest/download/clusterlink.sh | sh -
    ```

2. Verify the installation:

    ```sh
    clusterlink --version
    ```
